Cloud Service Explorer

Major cloud providers offer a vast array of services. Here are some of the most fundamental ones categorized by function.

AWS EC2

AWS

Provides scalable virtual servers in the cloud.

  • • Running web servers
  • • Hosting applications
  • • High-performance computing

Azure Virtual Machines

Azure

On-demand, scalable computing resources in Microsoft's cloud.

  • • Development and testing
  • • Running enterprise applications
  • • Disaster recovery

Google Compute Engine

GCP

High-performance virtual machines running on Google's infrastructure.

  • • Data analytics
  • • Machine learning workloads
  • • High-performance computing

Amazon S3

AWS

Object storage built to store and retrieve any amount of data.

  • • Backup and restore
  • • Data archiving
  • • Content distribution

Azure Blob Storage

Azure

Massively scalable object storage for unstructured data.

  • • Serving images/videos
  • • Backup and restore
  • • Big data analytics

Google Cloud Storage

GCP

Unified object storage for developers and enterprises.

  • • Data lakes
  • • Website hosting
  • • Backup and archive

Amazon RDS

AWS

Managed relational database service for MySQL, PostgreSQL, and more.

  • • Web applications
  • • E-commerce platforms
  • • Data warehousing

Azure SQL Database

Azure

Managed SQL database service in the cloud.

  • • Modern cloud applications
  • • Migration from on-premises
  • • Business intelligence

Google Cloud SQL

GCP

Fully managed database service for MySQL, PostgreSQL, and SQL Server.

  • • Web and mobile apps
  • • Enterprise applications
  • • Data analysis

Amazon VPC

AWS

Logically isolated section of the AWS Cloud where you can launch resources.

  • • Network isolation
  • • Hybrid cloud connectivity
  • • Multi-tier applications

Azure Virtual Network

Azure

Fundamental building block for your private network in Azure.

  • • Secure communication
  • • Internet access control
  • • Multi-site connectivity

Google Cloud VPC

GCP

Global private network for your Google Cloud resources.

  • • Multi-region deployments
  • • Secure hybrid connectivity
  • • Network segmentation
